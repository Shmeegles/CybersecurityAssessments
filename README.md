# Cybersecurity Assessments Portfolio

## Overview

This repository contains cybersecurity assessments and security-focused projects completed as part of my Bachelor of Science in Cybersecurity at Sacred Heart University.

These projects demonstrate hands-on experience with:

- Vulnerability Assessment
- Web Application Security Testing
- SQL Injection Analysis
- Password Auditing
- Linux Administration
- Log Analysis
- Network Security
- Security Reporting and Documentation

## Assessments Included

### SQL Injection Assessment
- Identified SQL injection vulnerabilities
- Performed database enumeration
- Demonstrated authentication bypass techniques
- Created remediation recommendations

### Burp Suite Intruder Assessment
- Conducted username enumeration testing
- Performed credential attack analysis
- Executed fuzzing exercises
- Documented security findings and recommendations

### Linux Security & Log Analysis
- Analyzed authentication logs
- Investigated failed login attempts
- Performed Linux security administration tasks
- Used command-line security tools for incident analysis

### Password Cracking & Password Audit
- Utilized Hashcat and John the Ripper
- Conducted password security assessments
- Analyzed password weaknesses
- Developed security recommendations based on findings
